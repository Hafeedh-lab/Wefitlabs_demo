# AI Quest Generator

AI-powered quest generation system for WeFit Labs that creates personalized, narrative-driven workout challenges.

## Project Structure

```
ai-quest-generator/
├── backend/          # Express.js API server
│   ├── src/
│   │   └── server.ts
│   └── package.json
├── frontend/         # React + Vite frontend
│   ├── src/
│   └── package.json
└── README.md
```

## Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

The server runs on http://localhost:3001

### Frontend

```bash
cd frontend
npm install
npm run dev
```

The frontend runs on http://localhost:5173
